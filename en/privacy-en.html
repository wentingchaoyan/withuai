1. Introduction

We respect the privacy of users of our child development support application (hereinafter referred to as "the App") and take the utmost care in protecting personal information. This Privacy Policy outlines how we handle personal information in the App.

2. Information We Collect

The App may collect the following information:

2.1 User Information

  • Phone number
  • Organization (nursery, therapy center, etc.)
  • Position

2.2 Child Information

  • Date of birth
  • Gender
  • Development status
  • Diagnostic information (optional)
  • Physical measurement data

2.3 Record Information

  • Development records
  • Daily records
  • Rehabilitation records
  • Video data
  • AI analysis results

3. Purpose of Information Use

Collected information is used for the following purposes:

  1. Provision and operation of the App
  2. Provision of development support services
  3. Provision of AI analysis features
  4. User support
  5. Service improvement
  6. Security maintenance

4. Information Management

1. We manage collected personal information appropriately and implement the following measures:

  • Access control settings
  • Data encryption
  • Security measures
  • Employee education

2. When outsourcing personal information handling, we provide appropriate supervision.

5. Information Sharing

We do not provide personal information to third parties except in the following cases:

  1. With user consent
  2. When required by law
  3. When necessary for the welfare of the child
  4. When sharing minimum necessary information within the scope of the App's usage purposes

6. Data Retention Period

  1. Personal information is retained only for the period necessary to achieve the purpose of use.
  2. When an account is deleted, related personal information is also appropriately deleted.

7. User Rights

Users have the following rights:

  1. Request for disclosure of personal information
  2. Request for correction or deletion of personal information
  3. Request for suspension of personal information use
  4. Request for data export

8. Security Measures

We implement the following security measures:

  1. Communication encryption using SSL/TLS
  2. Access control implementation
  3. Regular security audits
  4. Incident response planning

9. Cookie Usage

The App may use cookies for the following purposes:

  1. User authentication
  2. Usage analysis
  3. Service improvement

10. Privacy Policy Changes

  1. We may change this Privacy Policy as necessary.
  2. Changes to the Privacy Policy will take effect upon notification on the App.
